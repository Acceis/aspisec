Overview

This Ruby module contains all Aspisec modules. Each Aspisec module is a Ruby class inherinting the Aspisec::Module base class. The "Modules" module also contains methods to manage all Aspisec modules.

def self.list_modules

def self.modules(conf: nil, logger: nil)
  cnf = conf || Aspisec::Config.new(logger).conf
  list_modules.map { |c| Aspisec::Modules.const_get(c).new(cnf, logger:) }
end
